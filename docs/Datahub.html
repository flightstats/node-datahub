createGroupCallback(config)

Create a group callback.
Parameters:
Name Type Description
config Object configuration details for new group callback
Properties
Name Type Attributes Default Description
name string group callback name
channelName string the channel name to monitor for new items
callbackUrl string the fully qualified location to receive callbacks from the server
parallelCalls number <optional>
 1 number of callbacks to make in parallel
startItem string <optional>
 fully qualified item location where the callback should start from
paused boolean <optional>
 false When true, this will pause a group callback
batch string <optional>
 'SINGLE' 'SINGLE' will return each item by itself, 'MINUTE' will return each minute's worth of data in the channel
heartbeat boolean <optional>
 false false for SINGLE, or true for MINUTE batches that will always have a heartbeat, a callback which identifies the end of a minute period
maxWaitMinutes number <optional>
 1 the maximum amount of time between retry attempts to the callbackUrl
ttlMinutes number <optional>
 0 if greater than 0, the hub will not attempt to send an item which is older than the ttl
